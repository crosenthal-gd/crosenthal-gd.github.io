FetchResponse

A Promise that resolves to the data returned by the URL or an error if the request failed
For successful responses, the returned value depends on the content type of the response
  • application/json - The json value of the response is returned
  • text/* - Text value is returned
  • Any other content type - The original response is returned as is. Its the responsibility of the calling function to read the response based on content type
if shouldIncludeResponseAttributes is true, the resolved value contains the data as well as other response attributes with the following structure { data:, status:, statusTest:, headers:} For requests that fail, an error of type FetchError is thrown
Type:
  • Promise.<(Object|FetchError)>
